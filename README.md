# covAC-simulation-study-2024
Data analysis files belonging to a simulation study on detecting cultural transmission and sibling interaction using polygenic scores which is pre-printed here: https://www.researchsquare.com/article/rs-5461256/v1 and has since been accepted at Behavior Genetics.
 
# R Project:
**covAC-simulation-study-2024.Rproj** - R Project of the repository
 
# R Files:
**create-datasets-final-version.R** - Creates simulated data sets using the corresponding functions of gnomesims.
**plots-submission.R** - Creates plots based on the data files.
 
# CSV Files:
**paper_mx_estimates.csv** - OpenMx-based parameter estimates.
**paper_mx_power.csv** - OpenMx-based power.
**paper_dz_estimates.csv** - Parameter estimates for simulations of the DZ-only section.
**paper_dz_power.csv** - Power for simulations of the DZ-only section.
**sample_params.csv** - Parameter estimates for simulations based on sample size.
**sample_power.csv** - Power for simulations based on sample size
 
# Folders:
**Appendix** - Analyses and results found in the appendix.
**Archive** - Messy archive files.
**Figures** - JPG figures found in the paper.
**BioPsy Colloquium** - Code files and figures based on questions at the BioPsy Colloquium.
